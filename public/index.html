Cross-Site Scripting (XSS)

XSS attacks inject malicious scripts into web pages viewed by other users. Test common XSS payloads to see how Cloudflare WAF protects against them.

Example Payloads:

<script>alert('XSS')</script> <img src=x onerror=alert('XSS')> javascript:alert('XSS')

SQL Injection

SQL injection attacks attempt to manipulate database queries by injecting malicious SQL code. Test various SQLi techniques against WAF protection.

Example Payloads:

' OR '1'='1 '; DROP TABLE users; -- 1' UNION SELECT * FROM users--

Remote Code Execution (RCE)

RCE attacks attempt to execute arbitrary code on the server. Test command injection and code execution payloads against WAF filters.

Example Payloads:

; ls -la $(whoami) `cat /etc/passwd`
